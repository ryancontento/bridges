# Is the bridge up?

Run Client Side:
```bash
$ cd bridges/client
$ npm install
$ npm run start
```

Run Server Side:
```bash
$ cd bridges/server
$ cp .env.example .env
$ vim .env
$ yarn install
$ yarn start
```


# API

### `GET` `/capefear`
> Get status of the Cape Fear Bridge

### `GET` `/isabella`
> Get status of the Isabella Holmes Bridge
